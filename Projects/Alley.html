Forest Bubble

An animation project created in Unity, depicting a serene forest scene turning into fire scene when the bubble pop.

Type: Game Design / Interactive Media

Time: 2025

Process & Concept

This project explores the intersection between nature and digital play. Users cultivate flowers that grow and transform into bursts of color, creating an evolving ecosystem of visual energy. Built using p5.js, the project emphasizes generative interaction and visual rhythm.
